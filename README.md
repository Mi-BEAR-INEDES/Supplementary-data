# Supplementary data
_ Absolute frequency by Clitellata species. This figure summarizes the absolute frequency of 23 earthworm species, based on 465 individual records reported in the literature from sampling sites located in the provinces of Buenos Aires and Córdoba, Argentina. 

The “simple rules” are visualized using network graphs that characterize the relationships among the variables involved in the extracted rules. In these graphs, rules are depicted as circles connected by arrows to the associated variables, following a directional flow from the left-hand side (lhs) to the right-hand side (rhs). The circle size indicates the support: the greater the support, the larger the corresponding circle. The color represents the lift, with red indicating the highest values.
